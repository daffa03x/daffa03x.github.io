Donasi Diferensia

Donasi Diferensia adalah sebuah platform berbasis web yang memungkinkan pengguna untuk melakukan donasi secara online dengan mudah dan aman. Proyek ini dirancang untuk mendukung berbagai program sosial atau kegiatan amal yang dikelola oleh Diferensia. Dengan integrasi payment gateway, pengguna dapat memilih metode pembayaran yang sesuai dan mendapatkan laporan transparan mengenai donasi yang mereka berikan.

  • Teknologi yang Digunakan
  • Laravel – Backend utama untuk mengelola logika bisnis, autentikasi, dan manajemen data
  • MySQL – Database untuk menyimpan informasi donasi, pengguna, dan transaksi
  • Git – Version control untuk pengelolaan kode sumber secara terstruktur
  • Next.js – Frontend framework berbasis React untuk membangun antarmuka yang cepat dan interaktif
  • Payment Gateway – Digunakan untuk memproses transaksi secara aman dan mendukung berbagai metode pembayaran

Donasi Raihmimpi

Donasi Raihmimpi adalah platform donasi online yang memungkinkan masyarakat untuk berkontribusi dalam berbagai program sosial dan amal yang dijalankan oleh Raihmimpi. Dengan sistem yang aman dan transparan, platform ini mempermudah pengguna untuk berdonasi melalui berbagai metode pembayaran dan mendapatkan laporan real-time terkait donasi mereka.

  • Teknologi yang Digunakan
  • Lumen – Micro-framework berbasis Laravel untuk backend yang ringan dan cepat dalam menangani API dan transaksi donasi
  • MySQL – Database untuk menyimpan informasi donasi, pengguna, dan transaksi
  • Next.js – Framework frontend berbasis React untuk pengalaman pengguna yang cepat dan responsif
  • Google Cloud – Server hosting berbasis cloud untuk memastikan skalabilitas, keamanan, dan kecepatan akses tinggi
  • Git – Version control untuk memastikan pengelolaan kode sumber yang aman dan terstruktur
  • Payment Gateway – Digunakan untuk memproses pembayaran secara aman dan mendukung berbagai metode transaksi
  • Watzap – Integrasi WhatsApp untuk komunikasi langsung dengan donatur dan notifikasi transaksi otomatis
  • TikTok Pixel, Meta Pixel, Facebook Pixel – Digunakan untuk analitik dan pelacakan performa kampanye pemasaran donasi
